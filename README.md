# Foodies 🍽️ – Culinary Social Media Platform

**Foodies** is a full-stack social media application designed for culinary enthusiasts to share dishes, connect with peers, and discover new recipes. Built during the **Programming Applications and Frameworks (IT3030)** course at **SLIIT**, the platform bridges the gap between traditional social networking and food-centric community engagement.

---

## 🔍 Project Overview
Developed in 2023, this platform follows the **Model-View-Controller (MVC)** architecture. It utilizes a **Spring Boot** backend to handle complex social interactions and a **React** frontend for a responsive user experience.

## ⚙️ Core Modules

### 🔐 User Management 
Developed the core security and identity layer of the application:
* **Authentication:** Engineered secure REST API endpoints for User Registration, Login, and Logout.
* **Profile Management:** Implemented full CRUD functionality for user profile creation and editing.
* **Session Handling:** Integrated session-based state management to enhance project security.
* **Testing:** Conducted rigorous API validation and debugging using Postman to ensure endpoint stability.

### 📝 Post & Engagement System 
* **Content Lifecycle:** Supported the full lifecycle (CRUD) of culinary posts, from creation to removal.
* **Social Interaction:** Backend logic for nested comments and replies, allowing community feedback on shared dishes.
* **Performance:** Focused on optimizing database queries to maintain high application responsiveness.

### 🔔 Notification Service
* **Real-time Feedback:** Managed the delivery and tracking of notifications for social interactions.
* **Status Tracking:** Users can mark notifications as read/unread or remove them from their dashboard.
